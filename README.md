## ⚙️ System Specs & Software
- **OS:** macOS Sequoia 15
- **WM:** [Yabai](https://github.com) (Tiling Window Manager)
- **Hotkey Daemon:** [skhd](https://github.com)
- **Terminal:** Kitty
- **Status Bar:** SketchyBar
- **CLI Tools:** `btop` (System Monitor), `cava` (Audio Visualizer), `neo-matrix`
